# GoodEval
Developer: Blake Thollaug

Description: The GoodEval API can be used to remotely evaluate arithmetic expressions and calculate a multitude of special functions. It takes a JSON format object containing a formatted problem string and a boolean value to toggle logs and returns an object with the recieved problem string, an answer string, and the logs if they were toggled on.

start server: python3 manage.py runserver 3000
new app : python3 manage.py startapp app_name
